# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog and this project follows Semantic Versioning where practical.

---

## [1.0.0] - 2026-07-13

### Added

- Passive subdomain enumeration using Sublist3r
- Passive subdomain enumeration using Subfinder
- Active brute-force enumeration using PureDNS
- Alternate hostname generation using AltDNS
- Validation of generated hostnames using MassDNS
- Automatic merging and deduplication of results
- Organized output directory for each target
- Optional generation of `final.txt`
