# Security Policy

## Reporting a Vulnerability

If you discover a security issue related to this repository, please open a GitHub Issue or contact me directly with the details.

Please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

I will review reports as soon as possible and work toward an appropriate resolution.

## Supported Versions

This project is provided as a personal research tool and is maintained on a best-effort basis. Security updates will be applied when necessary.
